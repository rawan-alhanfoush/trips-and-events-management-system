# Trips and Events Management System

## Overview

This project was developed as part of a university team project. It is a Java desktop application that helps users manage trips and events. The system allows users to create an account, book trips, reserve events, choose additional services, make payments, receive notifications, and leave reviews.

The application uses Java for the desktop interface, MySQL for database management, and Maven for dependency management.

---

## Features

- User registration and login
- CAPTCHA verification
- Trip booking
- Event booking
- Additional services
- Payment system
- Notifications
- Reviews and ratings
- Customer dashboard
- Admin dashboard
- MySQL database integration

---

## Technologies

- Java
- Java Swing
- MySQL
- Maven
